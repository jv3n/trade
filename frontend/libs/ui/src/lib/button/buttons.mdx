# Button

Material's `<button mat-*>` restyled via the lib (`libs/ui/src/lib/button/button.scss`), shipped with two directives carried by `StbButtonModule` :

- **`[stbSize]`** — `xs | sm | md | lg` swaps the MDC container-height + label-text-size tokens.
- **`[stbSpinnerEnd]`** — applied to a `<mat-spinner>` inside a button, pushes it to the end of the row (trailing position) instead of the leading-icon slot.

## How to use

<Source
  language="ts"
  code={`import { StbButtonModule } from '@portfolioai/ui';

@Component({
  imports: [StbButtonModule],
  template: \`
    <button mat-flat-button (click)="save()">Save</button>
    <button mat-flat-button stbSize="lg" (click)="signIn()">Sign in with Google</button>
    <button mat-icon-button stbSize="xs"><mat-icon>close</mat-icon></button>
  \`,
})`}
/>

## Variants

<Markdown>
{`
| Selector              | Role                       | Use it for                                            |
| --------------------- | -------------------------- | ----------------------------------------------------- |
| \`mat-flat-button\`     | Filled, no elevation       | **Primary CTA** (Save, Generate, Confirm…).            |
| \`mat-stroked-button\`  | Outlined                   | **Secondary CTA** (Cancel, Edit…).                     |
| \`mat-button\`          | Text only, low emphasis    | Tertiary actions, inline toolbar buttons.              |
| \`mat-icon-button\`     | Square icon, no label      | Toolbar / overflow / quick toggles.                    |
| \`mat-raised-button\`   | Filled + elevation         | Rare ; same palette as \`mat-flat-button\`.            |
`}
</Markdown>

## Sizes

`sm` is the design-system default — same height as a button without `[stbSize]`.

<Markdown>
{`
| Token            | Container height | Label size | Typical use                              |
| ---------------- | ---------------- | ---------- | ---------------------------------------- |
| \`stbSize="xs"\`   | 24 px            | 0.75 rem   | Inline chip-row actions, table row icons. |
| \`stbSize="sm"\`   | 32 px (default)  | 0.8125 rem | Dialogs, dense toolbars.                  |
| \`stbSize="md"\`   | 38 px            | 0.875 rem  | Default CTAs in the journal / settings.   |
| \`stbSize="lg"\`   | 44 px            | 0.95 rem   | Hero CTAs (login, primary action page).   |
`}
</Markdown>

## Composing icon + label

Drop the icon and the label as direct siblings inside the button — Material projects the icon into the leading slot, label fills the default slot :

<Source
  language="html"
  code={`<button mat-flat-button>
  <mat-icon>auto_awesome</mat-icon>
  <span>Generate</span>
</button>`}
/>

**Caveat** (Material 21+) : if the icon is wrapped in a `@if` / `@else` block, each branch **must contain a single root node** matching the icon-slot selector. Otherwise the compiler raises `NG8011`. Split the conditional :

<Source
  language="html"
  code={`<button mat-flat-button [disabled]="loading()">
  @if (loading()) {
    <mat-spinner diameter="14"></mat-spinner>
  } @else {
    <mat-icon>refresh</mat-icon>
  }
  <span>{{ loading() ? 'Loading…' : 'Refresh' }}</span>
</button>`}
/>

## Loading state

Place the spinner where the icon would naturally go (leading), OR at the end via `[stbSpinnerEnd]` :

<Source
  language="html"
  code={`<!-- Leading (replaces the icon) -->
<button mat-flat-button disabled>
  <mat-spinner diameter="16"></mat-spinner>
  <span>Saving…</span>
</button>

<!-- Trailing -->
<button mat-flat-button disabled>
  <span>Saving</span>
  <mat-spinner stbSpinnerEnd diameter="16"></mat-spinner>
</button>`}
/>
