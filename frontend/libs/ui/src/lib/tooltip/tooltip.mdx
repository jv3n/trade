# Tooltip

`matTooltip` directive — hover / focus reveals a small bubble. `matTooltipPosition` controls placement (`above` / `below` / `left` / `right`).

## How to use

<Source
  language="ts"
  code={`import { StbTooltipModule } from '@portfolioai/ui';

@Component({
  imports: [StbTooltipModule],
  template: \`...\`,
})`}
/>
