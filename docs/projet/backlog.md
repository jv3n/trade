## Conventions de labels

Trois dimensions, combinables sur une issue :

| Dimension | Labels |
|-----------|--------|
| **Priorité** | `prio:P1` 🔴 · `prio:P2` 🟡 · `prio:P3` 🟢 |
| **Module** | `module:account` · `module:journal` · `module:stats` · `module:lexicon` |
| **Type** | `enhancement` · `bug` · `tech-debt` · `documentation` · `question` |

## Filtres utiles
