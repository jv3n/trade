# Red flags — fiche de révision

> Deux phases :
> - **Before** = analyse premarket avant 9:30 ET → dégrade le **sizing** (taille de position).
> - **After** = signaux pendant la séance → raccourcissent le **temps à la fermeture** de la position (sortie anticipée).

---

## Before — phase d'analyse premarket

Quand un titre matche la checklist [pattern](./pattern.md), on compte ses red flags pour décider la taille.

**Sizing** : 0 RF → full · 1 RF → /2 · 2 RF → /4

| Red flag | Pourquoi c'est un KO |
|----------|----------------------|
| **Float < 1M shares** | Trop *squeezable* — quelques acheteurs te liquident. |
| **Gap up < 45 %** | Pas mûr — pas assez de déconnexion à corriger. |
| **Gap up > 250 %** | Trop violent — ne retombe plus, le momentum prend le relais. |
| **Tendance marché défavorable** | Pas de setup propre sur le screener → on attend. |
| **Cours proche de $1** | *Survival mode* (anti-delisting Nasdaq) → reverse split imminent ou erratique. |
| **Institutional ownership ≥ 20 %** | Gros porteurs défendent le cours → le pump devient un vrai squeeze. |
| **Daily volatile sur historique** | Le titre fait des gaps comme ça souvent → setup pas unique. |
| **SSR actif** | *Short Sale Restriction* (chute ≥ 10 % la veille). Short uniquement à l'*uptick* → entrée ratée, exécution dégradée, risk/reward effondré. |
| **Hot sector** | Secteur entier qui porte le titre (ex. COVID 2020) → logique macro, pas spéculative. |
| **Volume la veille** | L'histo de la semaine doit être neutre. Volume préalable = signal déjà en cours, pas un *spike* isolé. |

---

## After — pendant la séance (post-open)

Signaux qui apparaissent **après 9:30 ET** et déclenchent une sortie plus tôt que prévu. **Action par défaut = fermeture anticipée** (partielle ou totale) ; cas avec sizing/timing calibré listés en dur.

| Signal | Sens / action |
|--------|---------------|
| **Push > +40 % à l'open** | **No go** — on n'entre pas du tout sur le trade. |
| **New high après 10:00 ET** | Coupe la position de **/2**. |
| **Reste dans le vert passé 11:00 ET** | Le retour à la baisse ne vient plus → ferme. |
| **Triple top avec volume faible** | Acheteurs essoufflés mais sans volume le breakdown est lent → prudence. |
| **New low puis rebond** | *Dip buyers* présents → le short risque de se faire écraser. |
| **Tient en hauteur sans volume** | Manipulation probable (gros trader derrière qui tient le cours). |
| **Rebond depuis une chandelle d'écrasement** | Les bears n'ont pas tenu le niveau bas. |
| **Float > 30M qui bouge fort** | Anormal pour ce float → data peut-être stale, méfiance. |
